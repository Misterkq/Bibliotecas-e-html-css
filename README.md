# 🌐 Estudos Full-Stack: Flask, HTML & CSS

Repositório dedicado ao meu aprendizado em desenvolvimento web completo, conectando interfaces visuais a servidores back-end.

## 📂 Projetos Atuais

### 🧪 Projeto com Flask
Meu primeiro contato com frameworks web em Python.
- **Funcionalidades:** Criação de rotas, renderização de templates HTML e organização de pastas (`/templates`, `/static`).
- **Aprendizado:** Entender como o servidor recebe uma URL e decide o que mostrar na tela.

### 🎨 Interfaces Web (Front-end)
- **Tela de Login:** Uma interface moderna criada com HTML5 e CSS3.
- **Foco:** Estilização, layout responsivo e usabilidade.

## 🛠️ Tecnologias e Ferramentas
- **Back-end:** Python & Flask
- **Front-end:** HTML5 & CSS3
- **Ambiente:** VS Code & Git

## 🚧 Em Construção
Este repositório reflete minha evolução. Atualmente estou focado em entender melhor o CSS e como integrar bancos de dados SQLite nos meus projetos Flask.

---
*Nota: Alguns conceitos de CSS e rotas foram desenvolvidos com apoio de IA para acelerar o entendimento de problemas específicos.*# 🌐 Estudos Full-Stack: Flask, HTML & CSS
